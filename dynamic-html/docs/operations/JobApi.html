This is the API

getJoblist

Get Job list. If username is defined in parameters, will retrieve joblist that user posted. If undefined, will retrieve all joblists.

URL

https://virtserver.swaggerhub.com/SJhumili/JobArchiveAPI/1.0.0/job

HTTP Method

GET

Response Type

List

Parameters

  • Header: username String

    The name that needs to be fetched. Use user1 for testing.
